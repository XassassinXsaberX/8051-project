## 參考接線方式
該紅色的USB開發板(採用PDIUSBD12晶片)與8051開發板TE-8051A(採用AT89S52 IC)  
的接線方式可參考如下  
D0～D7 <--------------> P2.0～P2.7  
 GND <--------------> GND  
 A0  <--------------> P3.5/T1  
 WR  <--------------> P3.6/WR  
 RD  <--------------> P3.7/RD  
 INT <--------------> P3.2/INT0
